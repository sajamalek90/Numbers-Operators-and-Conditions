Numbers, Operators, and Conditions

Number

JavaScript numbers can be integers or floating-point numbers. They are stored as 64-bit floating-point numbers.

let num1 = 42; // Integer
        let num2 = 3.14; // Floating-point number

Output: A screenshot of JavaScript code declaring numbers.

Number Methods

JavaScript provides several methods to work with numbers, such as toString(), toFixed(), and parseInt().

let num = 123.456;
        console.log(num.toFixed(2)); // 123.46
        console.log(num.toString()); // "123.456"

Output: A screenshot of the browser console showing the result of number methods.

Math Object

The Math object provides mathematical functions and constants, such as Math.PI, Math.sqrt(), and Math.random().

console.log(Math.PI); // 3.141592653589793
        console.log(Math.sqrt(16)); // 4
        console.log(Math.random()); // Random number between 0 and 1

Output: A screenshot of the browser console showing the result of Math object methods.

Create a program that generates a random number between 1 and 100 and rounds it to the nearest integer.

let randomNum = Math.random() * 100;
        let roundedNum = Math.round(randomNum);
        console.log(roundedNum);

String Methods

JavaScript provides several methods to manipulate strings, such as toUpperCase(), toLowerCase(), and substring().

let str = "Hello, World!";
        console.log(str.toUpperCase()); // "HELLO, WORLD!"
        console.log(str.substring(0, 5)); // "Hello"

Output: A screenshot of the browser console showing the result of string methods.

Comparison Operators

Comparison operators are used to compare two values. They return a boolean value (true or false).

console.log(10 > 5); // true
        console.log(10 == "10"); // true (loose equality)
        console.log(10 === "10"); // false (strict equality)

Output: A screenshot of the browser console showing the result of comparison operators.

Logical Operators

Logical operators are used to combine multiple conditions. They include && (AND), || (OR), and ! (NOT).

console.log(true && false); // false
        console.log(true || false); // true
        console.log(!true); // false

Output: A screenshot of the browser console showing the result of logical operators.

If Condition

The if statement is used to execute a block of code if a condition is true.

let age = 18;
        if (age >= 18) {
            console.log("You are an adult.");
        }

Output: A screenshot of the browser console showing the result of an if condition.

Nested If Condition

Nested if statements are used to check multiple conditions.

let age = 18;
        let hasLicense = true;
        if (age >= 18) {
            if (hasLicense) {
                console.log("You can drive.");
            }
        }

Output: A screenshot of the browser console showing the result of a nested if condition.

Conditional Ternary Operator

The ternary operator is a shorthand for the if-else statement.

let age = 18;
        let status = (age >= 18) ? "Adult" : "Minor";
        console.log(status); // "Adult"

Output: A screenshot of the browser console showing the result of a ternary operator.

Nullish Coalescing Operator

The nullish coalescing operator (??) returns the right-hand operand if the left-hand operand is null or undefined.

let value = null;
        let result = value ?? "Default Value";
        console.log(result); // "Default Value"

Output: A screenshot of the browser console showing the result of the nullish coalescing operator.

Switch Statement

The switch statement is used to perform different actions based on different conditions.

let day = "Monday";
        switch (day) {
            case "Monday":
                console.log("Start of the workweek");
                break;
            case "Friday":
                console.log("End of the workweek");
                break;
            default:
                console.log("Midweek");
        }

Output: A screenshot of the browser console showing the result of a switch statement.

let day = "Saturday";
